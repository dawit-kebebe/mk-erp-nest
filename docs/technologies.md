## **Technologies**

* **Framework:** [NestJS](https://nestjs.com/) (Node.js framework for building efficient, scalable Node.js server-side applications)
* **ORM:** [TypeORM](https://typeorm.io/)
* **Database:** MySQL (Recommended), PostgreSQL (Alternative)
* **Validation:** [Zod](https://zod.dev/) (for environment variables and DTOs)
* **Package Manager:** pnpm
* **Language:** TypeScript
